## 开始

```
pnpm run bootstrap, Don't use cnpm install.
```

### 启动开发服务器 并自定义端口

```
pnpm run dev --port = 4396
```

### 构建预发布环境

```
pnpm run build:stage
```

### 构建生产环境

```
pnpm run build
```

### 预览发布环境效果

```
pnpm run preview
```

### 预览发布环境效果 + 静态资源分析

```
pnpm run report
```

### 压缩 svg 图标

```
pnpm run svgo
```
